# Obra Kasi Games — Website

Marketing site for Obra Kasi Games, a Filipino game studio. Built with React + TypeScript + Vite, styled by a Claude Design design system rooted in Philippine folklore and pattern.

## Tech stack

- **React 19** + **TypeScript** (strict)
- **Vite** — dev server & build
- **react-router-dom** — client-side routing
- **lucide-react** — icons
- **ESLint** (typescript-eslint) — linting
- Design tokens & motifs as plain CSS (custom properties)

## Getting started

```bash
npm install
npm run dev
